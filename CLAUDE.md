# PBL-Gaokao System — Project Instructions

## What This Project Is

A comprehensive PBL (Project-Based Learning) framework for ages 2-18 aligned with China's 高考 system, built for the AI era, focused on preserving 主体性 (agency) and intrinsic motivation.

## Current Phase

**Structure + Research** — We are NOT designing specific curriculum yet. We are:
1. Building the structural framework
2. Researching underlying knowledge requirements (what 高考 actually demands)
3. Mapping motivation possibilities (what kinds of relevance/projects/problems could work)
4. Preparing for small experiments

## Key Principles

- **80/20**: 80% effort on motivation creation, 20% on scaffolding
- **Non-zero-sum**: Liberation worldview, not competition for scarce spots
- **Journaling as instrument**: Observation journals (Presencing-inspired) are core
- **"我可以..."**: Agency is the measurable outcome at every stage
- **AI as amplifier**: Never replaces student agency

## File Structure

- `framework/` — Age-stage documents (philosophy + 5 stages)
- `mapping/` — Test-to-problem transformation methodology + subject examples
- `motivation/` — Motivation architecture, 主体性 model, AI integration
- `assessment/` — Assessment design + 高考 readiness strategy
- `risk/` — Risk analysis and mitigations
- `missions/`, `specs/`, `verification/` — Three-role workflow artifacts

## Working Language

English primary, Chinese terms preserved where they carry meaning:
高考, 主体性, 语文, 应试, 中考, 文言文, 跨越式发展
