## What This Project Is

A comprehensive PBL (Project-Based Learning) curriculum framework for ages 2-18 that fully aligns with China's 高考 system while preserving intrinsic motivation, building 主体性 (agency), and preparing students for the AI era. The core mechanism transforms every exam knowledge point into a real-world problem worth solving.

## Current Priorities

- [ ] Complete the age-stage framework (5 stages, 2-18)
- [ ] Build the test-to-problem mapping methodology with concrete examples
- [ ] Document motivation architecture and 主体性 cultivation model
- [ ] Design assessment approach that bridges PBL and 高考 readiness
